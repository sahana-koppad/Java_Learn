# Java Learn

## Overview

This repository contains Java programs focused on Object-Oriented Programming (OOP) and Java Collections Framework. It is designed for learning core concepts, improving coding practice, and preparing for technical interviews.

---

### Object-Oriented Programming (OOP)

- Classes and Objects  
- Encapsulation (Data hiding, Getters and Setters)  
- Inheritance (Single, Multilevel, Hierarchical)  
- Polymorphism (Method Overloading and Overriding)  
- Abstraction (Abstract classes and methods)  
- Interfaces (Multiple inheritance using interfaces)  
- Constructors  
- this and super keywords  
- static and final keywords  
- Access Modifiers  

---

### Java Collections Framework

- List: ArrayList, LinkedList, Vector, Stack  
- Set: HashSet, LinkedHashSet, TreeSet  
- Map: HashMap, LinkedHashMap, TreeMap  
- Queue: PriorityQueue, Deque
